- phase: Phase 1 handoff preparation

## 1. Required Deliverables
The Phase 1 handoff package must include:
- frozen architecture specification
- frozen Data Unit schema
- frozen Evidence schema
- frozen Agent API specification
- pilot case sample materials
- OpenClaw single-agent workflow configuration
- example AgentRequest JSON
- example AgentResult JSON
- example Data Unit data
- example Evidence data
- GOA minimum task chain definition
- known limitations and unfinished items list
