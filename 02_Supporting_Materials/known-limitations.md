# Phase 0 Known Limitations and Open Items

- phase: Phase 0
- status: draft

## 1. Current Limitations
- Only BEAM Plus Existing Buildings v3.0 is supported as authoritative framework.
- Only Management (MAN) is prioritized in Phase 0.
- Only MAN-02-01, MAN-02-02, MAN-02-03, and MAN-03-04 are included as pilot credits.
- B Agent and E Agent are reserved at interface level only.
- Cross-Agent Arbitration Engine is not implemented.
- Mapping Engine is not implemented.
- Platform UI is not included in Phase 0.
- Full certification scoring and rating are not included.
- OpenClaw runtime workflow rendering has not yet been verified.
- Pilot runnable verification has not yet been completed.
